<h1> Pet-Dada </h1> 
<div>
 반려동물을 가족처럼 여기는 이들을 위한 커뮤니티가 있으면 좋을 것 같다는 생각에 시작하게 되었음
</div>

<h3>첫 방문 시 사용법</h3>
<pre>
- localhost/member/login로 접속하게 되면, 로그인 창이 뜸
- 아이디가 없는 경우에는 회원가입을 해야하며 가입 후에는 이메일 인증을 거쳐야 로그인이 성공됨
</pre>

<h3>사이드바 사용법</h3>
<pre>

- 
</pre>

<h3>로그인 사용법</h3>
<pre>
- 로그인이 완료된 경우에는 사이드 바에 로그아웃이라 뜨고, 로그아웃 상태가 되면 로그인이라 뜸
</pre>
